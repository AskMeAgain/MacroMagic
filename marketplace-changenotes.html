Macro Magic

This plugin improves on the current Macro implementation of Intellij.
  • First release!
  • It adds an extensive list of edit features (split, duplicate, combine, extract, insert, rearrange etc.)
  • Import/Export macros to xml
  • Nested Macros
  • A nice GUI
  • Settingswindow
  • More util actions to move the cursor (SelectAnnotation, SelectClass, JumpToTop)
